# 即时空教室查询小程序SSM

## 前言

随着数字化校园的不断发展，即时空教室查询成为了高校中不可或缺的一项服务。为此，我们开发了这款基于SSM框架和微信小程序的即时空教室查询系统。旨在为广大师生提供便捷、高效的教室查询服务。

## 内容介绍

本项目是一款适用于高校的即时空教室查询小程序，用户可以通过微信小程序端查询到实时的空教室信息。后端采用Java语言，结合Spring、SpringMVC、MyBatis等主流框架，确保系统的高效稳定。前端则使用JS、Vue、CSS3和Uniapp等技术，实现用户友好的交互体验。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis，微信小程序
- 前端技术：JS、Vue、CSS3，Uniapp
- 开发工具：IDEA/Eclipse，Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
